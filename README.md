# Black Market Brewing Co. — Website Redesign Concept

Modern, sleek redesign mockup for [Black Market Brewing Co.](https://blackmarketbrew.com/) in Temecula, CA.

## Features
- **Hero** — Full-screen with live music emphasis, hours, and CTAs
- **Live Events** — Card grid with upcoming bands, dates, and genres
- **Tap List** — Categorized, filterable beer/wine/cider menu with ABV/IBU
- **Food Menu** — Wood-fired pizza showcase with prices and photos
- **Event Space** — 6,000 sqft private event booking section
- **Responsive** — Desktop, tablet, mobile optimized

## Tech Stack
- Next.js 15 + TypeScript
- Tailwind CSS
- Deployed on Vercel

## Getting Started
```bash
npm install
npm run dev
```
